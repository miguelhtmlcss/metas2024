HTML (HyperText Markup Language)

HTML é a linguagem padrão para criar páginas web. Ele estrutura o conteúdo da web através de marcações, permitindo que navegadores interpretem e exibam textos, imagens, vídeos, links e outros elementos de uma página web.

CSS (Cascading Style Sheets)

CSS é usado para definir a apresentação visual dessas páginas. Ele permite estilizar e formatar o conteúdo HTML, controlando aspectos como cores, fontes, layouts, e espaçamentos, contribuindo para a aparência e usabilidade das páginas web.
